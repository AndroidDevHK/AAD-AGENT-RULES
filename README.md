# AAD-AGENT-RULES 🤖📱

Welcome to the **AAD-AGENT-RULES** project! This repository contains a comprehensive set of rules, prompts, and guidelines tailored specifically for Android App Development (AAD) when using advanced AI coding assistants and agents.

## 🌟 Overview

The goal of this project is to streamline and standardize the Android development workflow when collaborating with AI tools. By utilizing these rules, developers can ensure higher-quality code, better architectural decisions, and avoid common pitfalls when relying on AI for code generation and refactoring.

## 🛠️ Supported AI Assistants

This rule set is designed to be highly effective with a wide variety of modern AI coding assistants, including but not limited to:

- **Anti Gravity**
- **Cursor**
- **Claude**
- **Codex**
- **OpenCode**
- *And many more...*
